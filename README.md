## 🤖 Introduction
This is my **interactive 3D portfolio website** built with **React**, **Three.js**, and **Tailwind CSS**.

Feel free to explore the code, check out the live version, and reach out through the contact section!

---

## ⚙️ Tech Stack

- React.js
- Three.js
- React Three Fiber
- React Drei
- EmailJS
- Vite
- Tailwind CSS
- Framer Motion

---

## 🔋 Features

- **Customizable 3D Hero Section** — Easily swap out and style the 3D desktop model.
- **Interactive Work Sections** — Animate with Framer Motion for engagement.
- **3D Skills Section** — Showcase tech stack in 3D using Three.js + React Three Fiber.
- **Projects & Testimonials** — Beautifully animated sections.
- **Contact Section** — 3D Earth model with EmailJS integration.
- **Dynamic Stars Background** — Procedurally generated with Three.js.
- **Consistent UI Animations** — Smooth transitions across pages.
- **Fully Responsive** — Works on all screen sizes.

## 🤸 Quick Start

### 🛠️ Prerequisites

- [Git](https://git-scm.com/)
- [Node.js](https://nodejs.org/en)
- [npm](https://www.npmjs.com/)

### 📦 Installation

```bash
git clone git@github.com:adrianhajdin/project_3D_developer_portfolio.git
cd project_3D_developer_portfolio
npm install
npm run dev
```
