## 🤖 Introduction
This is my **interactive 3D portfolio website** built with **React**, **Three.js**, and **Tailwind CSS**.

Feel free to explore the code, check out the live version, and reach out through the contact section!

---

## ⚙️ Tech Stack

This portfolio was crafted using the following tools and libraries:

- ⚛️ **React.js** – Building reusable UI components
- 🌐 **Vite** – Fast and modern frontend build tool
- 🎮 **Three.js** – Rendering 3D graphics in the browser
- 🧵 **React Three Fiber** – React renderer for Three.js
- 🪄 **React Drei** – Useful helpers for React Three Fiber
- 🎨 **Tailwind CSS** – Utility-first CSS framework for styling
- 🎥 **Framer Motion** – Smooth animations and transitions
- 📧 **EmailJS** – Sending emails directly from the contact form

---

## 🔋 Features

- 🖥️ **Customizable 3D Hero Section** — Easily swap out and style the 3D desktop model.
- 🧩 **Interactive Work Sections** — Animate with Framer Motion for better engagement.
- 🧠 **3D Skills Section** — Showcase your tech stack in 3D using Three.js + React Three Fiber.
- 📁 **Projects & Testimonials** — Beautifully animated and cleanly structured sections.
- 🌍 **Contact Section** — Includes an interactive 3D Earth model with EmailJS integration.
- 🌌 **Dynamic Stars Background** — Procedurally generated with Three.js for a cosmic vibe.
- 🎞️ **Consistent UI Animations** — Smooth transitions across pages for a polished UX.
- 📱 **Fully Responsive** — Looks great on desktops, tablets, and smartphones.

## 🤸 Quick Start

### 🛠️ Prerequisites

- [Git](https://git-scm.com/)
- [Node.js](https://nodejs.org/en)
- [npm](https://www.npmjs.com/)

### 📦 Installation

```bash
git clone git@github.com:adrianhajdin/project_3D_developer_portfolio.git
cd project_3D_developer_portfolio
npm install
npm run dev
```

#### Open http://localhost:5173 in your browser to view the project.
